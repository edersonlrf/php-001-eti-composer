PHP - Especializa TI - Composer
- Composer é um gerenciador de pacotes (dependências)
	https://getcomposer.org/
- Acesse a pasta do projeto pelo Prompt
- Crie o arquivo JSON
	composer init
	- "require" e "require-dev" são as dependências dos ambientes de produção e desenvolvimento.
- Atualize as dependencias do projeto
	composer update
	- O diretório "vendor" ficam as dependências do projeto
- Insira no projeto o "require" que ira trazer todas as dependencias da aplicação
	require_once './vendor/autoload.php';
- Defina a no "composer.json", em "require", a versão do PHP que o projeto executa
	- Se for uma versao menor, sera lancado um erro
- Defina as dependencias do ambiente de desenvolvimento em "require-dev"
	- Com "^" o composer pega no minimo a versao 4.3 ate a 5.0
- Não altere arquivos dos projetos dentro da pasta "vendor", pq quando for feito um "composer update" vc perdera as alteracoes
- Para pesquisar os pacotes
	https://packagist.org/
